# Nefli - Laboratorio 10

App Flutter estilo Netflix con datos de películas desde la API [OMDB](https://www.omdbapi.com/).

## Requisitos cubiertos

- ListViews, ListTiles, Cards
- Listas y Mapas
- Rutas con navegación nombrada
- Tema global estilo Netflix
- 3 tipos de letra (Bebas Neue, Roboto, Montserrat)
- Imágenes locales y en la nube

## Ejecutar

```bash
flutter pub get
flutter run
```

## API

La clave OMDB se configura en `lib/config/api_config.dart`.
